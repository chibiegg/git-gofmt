# gofmt Git Commit Hook

## Installation

1. ```$ cd your_project```
2. Save pre-commit with ```$ wget https://raw.githubusercontent.com/chibiegg/git-gofmt/master/pre-commit -O .git/hooks/pre-commit```
3. Mark pre-commit executable: ```$ chmod +x .git/hooks/pre-commit```
